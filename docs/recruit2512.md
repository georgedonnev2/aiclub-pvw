# 俱乐部招新启示-2512

为提高学生创新实践能力，依托华为ICT学院和华为智能基座，学院成立了 **昇腾鲲鹏AI创新俱乐部**。

## 首次招新说明

本次招新，是俱乐部成立后首次招新。有兴趣加入俱乐部的同学，请完成以下 4 个事宜：

1. 加入QQ群。
2. 放寒假之前，预约杨老师、董老师面聊。
3. 接受并完成 1-2 周的学习主题和内容。
4. 开学第一周，参加俱乐部招募面试。

## 一、任务目标

俱乐部将承接以下目标：

1. 对接“华为ICT大赛”（主要目标）
2. 参加“华为昇腾AI创新大赛”（2022年首届）
3. 参加“华为鲲鹏创新大赛”（2020年首届）
4. 支撑学院暑期开发板训练营
5. 支撑学院人工智能相关实验课（华为昇腾/鲲鹏开发板相关）
6. 支撑全体师生对华为昇腾/鲲鹏开发板的日常使用

## 二、组织架构

- 负责老师：杨金龙、董俊杰。
- 俱乐部成员（学生）。面向全院学生招募遴选，20人左右。
- 热烈欢迎感兴趣老师和学生，指导和参加俱乐部相关活动。

## 三、招募方法

俱乐部将采用以下方法招募成员：

- 感兴趣学生自愿报名。
- 报名学生接受1-2周的学习主题和内容。
- 学习完成后，报名学生参加俱乐部招募面试。
- 由俱乐部招募团队评分讨论后，确定入选名单。额满为止。

## 四、管理约定

俱乐部成员要遵守以下约定：

1. 每年度至少参加 3 个比赛的其中之一。
2. 轮班参加 3 个支撑事宜。
3. 每年度参加俱乐部公共活动和例会的次数，不少于总次数的 2/3。
